## 🔬 机制分类规则详解

### 规则1: 被动扩散

**判断条件:**
```python
IF (MW < 500) AND (TPSA < 90) AND (LogP > 0) AND (LogP < 5):
    mechanism = "passive"
```

**理化性质要求:**
- ✅ 分子量 < 500 Da
- ✅ 极性表面积 < 90 Ų
- ✅ 脂水系数 0-5 (适中)
- ✅ 符合Lipinski类药五原则

**典型特征:**
- 小分子、低极性
- 不依赖转运蛋白
- 可自由穿过细胞膜

**数据:** 4,271个化合物 (54.7%)

**例子:** 阿司匹林、咖啡因

---

### 规则2: 主动内排

**判断条件:**
```python
IF (MACCS[42] == 1) OR (MACCS[35] == 1):
    mechanism = "influx"
```

**MACCS子结构:**
- **MACCS 42**: 两个氨基基团连接在同一碳上
  ```
  N-C(-N)-  # 双氨基结构
  ```
  
- **MACCS 35**: 硫杂环
  ```
     S
    / \
  - C -
   /   \
  ```
  类似噻唑/硫杂环结构

**典型特征:**
- 高TPSA (>100 ��)
- 高分子量 (>450 Da)
- 多氢键受体
- 可能被SLC转运蛋白(氨基酸转运体)识别

**数据:** 385个化合物 (4.9%)

**例子:** 某些氨基酸类似物

---

### 规则3: 主动外排

**判断条件:**
```python
IF (MACCS[8] == 1) OR (MW > 500):
    mechanism = "efflux"
```

**MACCS子结构:**
- **MACCS 8**: 四元环杂原子
  ```
     O
    // 
   //   (beta-lactam)
    \
     N
  ```
  青霉素类抗生素的beta-lactam环

**典型特征:**
- 大分子 (>500 Da)
- 可能是P-gp等外排泵的底物
- 被主动泵出血脑屏障

**数据:** 1,220个化合物 (15.6%)

**注意:** B3DB中beta-lactam很少，所以这主要是基于分子量

---

### 规则4: 混合机制

**判断条件:**
```python
IF NOT (passive OR influx OR efflux):
    mechanism = "mixed"
```

**典型特征:**
- TPSA最高 (>110 Ų)
- 分子量较大
- LogP异常(很低或很高)
- BBB通透性最差

**数据:** 1,929个化合物 (24.7%)

## 📋 完整分类流程

```
输入: SMILES字符串
        ↓
    [计算理化性质]
    TPSA, MW, LogP, HBD, HBA, RotBonds
        ↓
    [生成MACCS指纹]
    167位结构指纹
        ↓
    [应用分类规则]
        ↓
    ┌─────────────────────────────────┐
    │ 满足Passive条件?                 │
    │ MW<500 AND TPSA<90 AND 0<LogP<5 │
    │                                 │
    │ YES → "passive" ──────┐         │
    │ NO                    │         │
    └───────────────────────┼─────────┘
                             │
                             ▼
    ┌─────────────────────────────────┐
    │ 含MACCS[42]或MACCS[35]?          │
    │ (双氨基或硫杂环)                  │
    │                                 │
    │ YES → "influx" ───────┐         │
    │ NO                    │         │
    └───────────────────────┼─────────┘
                             │
                             ▼
    ┌─────────────────────────────────┐
    │ 含MACCS[8]或MW>500?             │
    │ (beta-lactam或大分子)             │
    │                                 │
    │ YES → "efflux" ────────┐         │
    │ NO                    │         │
    └───────────────────────┼─────────┘
                             │
                             ▼
                    "mixed" (默认)
```

| 机制 | TPSA (Ų) | MW (Da) | LogP | HBA | 主要特征 |
|------|----------|---------|------|-----|----------|
| **Passive** | **<90** | **<500** | 0-5 | 低 | 小分子、低极性 |
| **Influx** | **>100** | **>400** | 中-高 | **高** | 双氨基、硫杂环 |
| **Efflux** | 变化 | **>500** | 变化 | 高 | Beta-lactam |
| **Mixed** | **最高** | **大** | 异常 | 最高 | 性质不适合 |

## 🔄 与文献方法的对比

### Cornelissen et al. 2022的方法

**实验数据集:**
1. BBB dataset - 内皮细胞单层实验
2. PAMPA dataset - 人工膜被动扩散实验
3. Influx dataset - SLC转运蛋白底物实验

5. CNS dataset - 临床CNS药物数据

**每个机制都基于实验数据训练独立模型。**

**数据来源:**
- ✅ BBB: B3DB实验数据

**优势:**

**劣势:**
