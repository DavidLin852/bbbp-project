## 功能介绍

在"Prediction"页面新增了第三个标签页 **"全部模型预测"**，可以一次性使用所有32个模型对分子进行预测，快速查看不同模型和数据集的预测结果对比。

---

## 使用方法

### 1. 进入页面
启动Streamlit平台后，进入 **Prediction** 页面，点击第三个标签 **"全部模型预测"**

### 2. 输入分子

有两种方式输入分子：

#### 方式1：直接输入SMILES
- 在文本框中输入SMILES，每行一个
- 最多支持5个分子

#### 方式2：上传CSV文件
- 上传包含SMILES列的CSV文件
- 最多读取前5行

### 3. 点击预测
点击 **"🚀 运行全部32个模型"** 按钮

### 4. 查看结果

#### 结果展示包括：

**① 预测汇总**
- 模型总数
- 预测为BBB+的模型数量
- 平均概率
- 标准差

**② 一致性分析**
- ✅ 所有模型一致预测为 BBB+
- ✅ 所有模型一致预测为 BBB-
- ⚠️ 模型预测分歧较大

**③ 各模型预测详情**
按数据集分组显示：
- **A 数据集**：8个模型
- **A,B 数据集**：8个模型
- **A,B,C 数据集**：8个模型
- **A,B,C,D 数据集**：8个模型

每个模型显示：
- 模型名称
- AUC
- BBB+概率
- 预测结果（BBB+/BBB-）

**④ 预测概率热力图**（仅单个分子时）
- 可视化展示所有32个模型的预测概率
- 颜色编码：红色(BBB-) → 绿色(BBB+)

## 使用场景

### 场景1：快速筛选
投一个候选分子，查看32个模型的整体判断：
- 如果所有模型都预测为BBB+ → 高置信度，可以推进
- 如果模型预测分歧大 → 需要进一步实验验证

### 场景2：模型选择
查看不同数据集训练的模型对同一分子的预测差异：
- A数据集模型预测 vs A,B,C,D数据集模型预测
- SMARTS增强模型 vs 原始模型
- GAT模型 vs 传统ML模型

### 场景3：结果验证
对已知活性的分子进行预测，验证模型的一致性和可靠性

## 功能特点

### ✅ 优点
1. **一次性对比32个模型** - 无需逐个模型测试
2. **智能一致性分析** - 自动判断模型预测是否一致
3. **按数据集分组** - 清晰展示不同数据集模型的预测
4. **可视化热力图** - 直观展示预测概率分布
5. **支持批量分子** - 最多5个分子同时预测
6. **导出完整结果** - CSV格式方便后续分析

### ⚠️ 限制
1. 最多支持5个分子（避免计算时间过长）
2. 预测时间取决于分子数量（32个模型 × N个分子）
3. 需要2-3分钟完成所有预测

|---------|---------|

| 3个分子 | 1-2分钟 |
| 5个分子 | 2-3分钟 |

## 结果解读示例

### 示例1：高置信度BBB+
```
分子: CCO (乙醇)
✅ 所有模型一致预测为 BBB+
平均概率: 0.956
标准差: 0.023
```
**解读**: 32个模型中有32个预测为BBB+，平均概率高且标准差低，结果可信度高

### 示例2：模型分歧
```
分子: CCN (乙胺)
⚠️ 模型预测分歧较大
平均概率: 0.523
标准差: 0.187
```
**解读**: 32个模型预测不一致（例如18个BBB+, 14个BBB-），标准差大，建议谨慎对待或进行实验验证

### 支持的32个模型

| 数据集 | RF+SMARTS | XGB+SMARTS | LGBM+SMARTS | GAT |
|--------|-----------|------------|-------------|-----|
| A | ✓ | ✓ | ✓ | ✓ |
| A,B | ✓ | ✓ | ✓ | ✓ |
| A,B,C | ✓ | ✓ | ✓ | ✓ |
| A,B,C,D | ✓ | ✓ | ✓ | ✓ |

### 性能优化
- 使用 `@st.cache_data` 缓存模型信息
- 进度条实时显示预测进度
- 异常处理：单个模型失败不影响整体
