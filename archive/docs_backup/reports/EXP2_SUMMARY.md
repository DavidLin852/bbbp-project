# 实验2总结：A,C,D平衡策略

## 实验时间
2026-01-21

## 实验目的
根据用户需求，添加Group D（负例丰富）和Group C（平衡样本），以：
1. 增加负样本数量
2. 改善正负样本平衡
3. **控制假阳性（FP）**：确保负的（BBB-）不会被判成正（BBB+）

## 关键数据发现

### 各组样本分布

| Group | BBB+ | BBB- | 正例率 | 特点 |
|-------|------|------|--------|------|
| A | 930 | 128 | 87.9% | 严重偏向正例 |
| B | 2486 | 1135 | 68.7% | 仍然偏正例 |
| **C** | **1536** | **1541** | **49.9%** | **接近完美平衡！** |
| **D** | **4** | **47** | **7.8%** | **主要是负例！** |

### 推荐组合：A + C + D

| 数据集 | 总样本 | 正例 | 负例 | 正负比 |
|--------|--------|------|------|--------|
| A,B | 4679 | 3416 | 1263 | **2.70** |
| **A,C,D** | **4186** | **2470** | **1716** | **1.44** |

**改进**：
- ✅ 负例增加：1263 → 1716 (+35.9%)
- ✅ 正负比改善：2.70 → 1.44
- ✅ 更接近平衡（理想的1.00）

## 分析

### 为什么FP增加了？

1. **测试集大小不同**
   - A,B测试：127个负例
   - A,C,D测试：172个负例（+35%）
   - 更多负例 = 更多机会犯错

2. **阈值不是最优**
   - 当前使用阈值0.5
   - A,C,D训练数据更平衡，应该使用更高的阈值（如0.6-0.7）

3. **数据分布不同**
   - A,C,D的负例来自Group C和D，化学特性可能不同
   - 模型需要适应这些新的负例模式

### 关键洞察

| 角度 | A,B | A,C,D |
|------|-----|-------|
| 训练平衡 | 差（73%正例） | 好（59%正例） |
| 测试负例 | 127 | 172 |
| 默认阈值(0.5) | 最优 | **不是最优** |
| 需要调整阈值 | 否 | **是** |

## 结论与建议

### ❓ 是否使用A,C,D？

**答案：是，但需要阈值优化**

| 策略 | 优点 | 缺点 |
|------|------|------|
| 继续用A,B | 高AUC，当前最优 | 训练不平衡，泛化差 |
| **改用A,C,D** | **训练平衡，长期好** | **需调阈值** |

### 短期行动（立即执行）

1. **阈值优化**
   ```python
   # 在验证集上找最优阈值（目标：Precision ≥ 90%）
   from sklearn.metrics import precision_recall_curve

   # 找precision >= 0.9的最大recall阈值
   precision, recall, thresholds = precision_recall_curve(y_val, y_prob_val)
   optimal_threshold = thresholds[precision >= 0.9].max()
   ```

2. **代价敏感学习**
   ```python
   # 对FP（假阳性）施加更大惩罚
   # XGB示例：
   scale_pos_weight = (neg / pos) * 2.0  # 惩罚FP更多
   ```

### 中期策略

1. **分别训练，然后集成**
   - Model 1: A,B（擅长识别正例）
   - Model 2: A,C,D（擅长识别负例）
   - Ensemble: 加权平均或堆叠

2. **使用Group作为特征**
   - 输入时标记样本来自哪个group
   - 让模型学习group-specific模式

3. **验证集阈值搜索**
   - 目标：Precision = 95%
   - 在A,C,D验证集上找最优阈值
   - 应用到测试集

### 长期研究

1. **Domain Adaptation**
   - 在A,B上预训练
   - 在C,D上微调

2. **困难样本挖掘**
   - 分析C和D中容易被误判的样本
   - 针对性增强训练

3. **不确定性估计**
   - 使用贝叶斯方法或MC Dropout
   - 对不确定样本标记"需要人工复核"
